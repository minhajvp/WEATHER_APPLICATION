# WEATHER_APPLICATION

Simple Weather App (React + Django + Open-Meteo API)

This is a simple weather application built using React (frontend) and Django REST Framework (backend).
It allows users to enter a city name, fetch its latitude, longitude, temperature, and wind speed, and stores the search in a database.

Search weather by city
- Fetch real-time temperature & wind speed
- Save request logs to database
- Glass Type UI design (React+CSS)
- CORS-enabled Django API
- Error handling for invalid city searches

  Tech Stack used:
  Frontend - vite React
  Backend - Django Rest Framework
  Database - SQLite
